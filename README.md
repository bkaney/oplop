Oplop
=====

This is a ruby implementation of Oplop http://code.google.com/p/oplop/

Install
=======

```
gem install oplop
```

Usage
=====

```ruby
Oplop.password(:master => "master-password", :label => "nickname")
```

CLI
===

This includes an `oplop` program.  Run `oplop --help` for more information.
